# 🎨 RE-SPACED - AI Interior Design Tool

Professional interior design transformation powered by artificial intelligence.

## 🚀 Features

- ✨ AI-powered interior design generation
- 🎯 Professional-grade results
- ⚡ Fast processing (30-60 seconds)
- 🎨 Ultra-modern luxury designs
- 📱 Fully responsive design
- 🔒 No watermarks

## 🛠️ Tech Stack

- HTML5 + Tailwind CSS
- Vanilla JavaScript
- Replicate API (adirik/interior-design model)
- Client-side processing

## 📋 Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- Replicate API token (free)

## 🔑 Setup

1. Get your free API token from [Replicate.com](https://replicate.com)
2. Replace `YOUR_API_TOKEN` in the code with your actual token
3. Open `index.html` in your browser
4. Upload a room photo and let AI transform it!

## 📖 How to Use

1. **Upload Image**: Click or drag-and-drop a room photo
2. **Generate**: Click "Generar Diseño" button
3. **Wait**: AI processes your image (30-60 seconds)
4. **Download**: Save your professional interior design

## 🎯 Supported Formats

- JPG/JPEG
- PNG
- Maximum file size: 10 MB
- Minimum resolution: 1200x800px
